# AI PDF Summarizer

AI PDF Summarizer is a web-based application that allows users to upload PDF documents and generate concise summaries using the Gemini API. The application extracts text from PDFs and leverages generative AI to provide meaningful insights in a simple and modern interface.

---

## Features

- Upload PDF files through a web interface  
- Extract text from PDF using pdfplumber  
- Generate summaries using Gemini API  
- Clean and responsive user interface  
- Fast and easy-to-use workflow  

---

## Tech Stack

- Python  
- Flask  
- Google Gemini API  
- pdfplumber  
- HTML, CSS
- ## Usage

1. Upload a PDF file  
2. Click on "Generate Summary"  
3. View the AI-generated summary on the screen  

---

## Notes

- Only text-based PDFs are supported  
- Large PDFs may be truncated for processing  
- Do not upload sensitive API keys to GitHub
